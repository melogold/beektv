# BeekTV

Purpose: Mobile IPTV app. The app will have the sole focus of streaming IPTV, based on M3U files that user's upload/servers they log into that send back M3U. It will avoid the Movies and TV series commonly found alongside IPTV in these types of apps.

## Features

- Stream IPTV from M3U sources
- EPG support
- Picture-in-picture support
- Background playback support (listen to IPTV stream with phone display off)
- Focus on fast M3U refreshes

## Development Guide

- Using Expo for mobile development
- Ask questions whenever unsure about direction
- Use proper Typescript (don't add `any` type when unsure, ask for direction if unsure)
- Focus on iOS but be flexible for multiplatform
- Use best Typescript and mobile development practices
